## Dependencias necessárias:

### Instalções para Frontend
```bash
npm install
npm install axios
npm install react-router-dom
npm install @lottiefiles/react-lottie-player
```

### Instalações para Backend

```Bash
npm install
npm install --save-dev nodemon
npm install express
npm install bcrypt
npm i jsonwebtoken
npm install dotenv --save
```
